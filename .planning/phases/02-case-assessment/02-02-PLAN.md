Create `app/services/limitation_service.py` implementing the research Pattern 3 exactly:

1. `from __future__ import annotations` as FIRST import (Python 3.14 requirement)

2. Define `LimitationStatus` enum:
   - WITHIN_PERIOD — more than 90 days remaining
   - WARNING — less than 90 days remaining
   - EXPIRED — basic 2-year period has passed
   - ULTIMATE_EXPIRED — 15-year ultimate period has passed
   - REQUIRES_LAWYER_REVIEW — tolling applied (minor or incapacity)

3. Define `LimitationResult` dataclass with fields:
   - status: LimitationStatus
   - basic_deadline: date | None
   - ultimate_deadline: date | None
   - days_remaining: int | None
   - warning_message: str | None
   - tolling_applied: bool = False
   - municipal_notice_required: bool = False

4. Implement `calculate_limitation()` function following the research code exactly:
   - Parameters: discovery_date, incident_date, is_minor, minor_dob, is_incapacitated, incapacity_end_date, is_municipal_defendant, today (optional, defaults to date.today())
   - Use `relativedelta(years=2)` for basic period (NOT timedelta)
   - Use `relativedelta(years=15)` for ultimate period
   - Use `relativedelta(years=18)` for minor age calculation
   - Import constants from ontario_constants: `LIMITATION_PERIOD_YEARS`, `ULTIMATE_LIMITATION_YEARS`, `MUNICIPAL_NOTICE_DAYS`
   - Actually use these constants in the relativedelta calls: `relativedelta(years=LIMITATION_PERIOD_YEARS)` etc.
   - Tolling logic for minors: if is_minor and minor_dob provided, calculate age_18 = minor_dob + relativedelta(years=18). If age_18 > basic_start, shift basic_start to age_18 and set tolling_applied=True.
   - Tolling logic for incapacity: if is_incapacitated and incapacity_end_date provided, if incapacity_end_date > basic_start, shift basic_start to incapacity_end_date and set tolling_applied=True.
   - When tolling_applied=True, ALWAYS set status to REQUIRES_LAWYER_REVIEW regardless of other status
   - When is_municipal_defendant=True, set municipal_notice_required=True and include a warning about the 10-day notice requirement
   - Warning thresholds: < 90 days = WARNING status; expired = EXPIRED; ultimate expired = ULTIMATE_EXPIRED
   - All warning messages should use plain language, not legal jargon

5. This service must have NO ORM imports — it is a pure Python function for easy unit testing. Only imports: `__future__`, enum, dataclasses, datetime, dateutil.relativedelta, and ontario_constants.
