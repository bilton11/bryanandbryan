**b) "Where am I?" Navigator:**

A horizontal step indicator (or vertical on mobile) showing all 8 stages. Uses Alpine.js:

```html
<div x-data="guideNav()" class="guide-nav">

```

Each stage is a clickable pill/button. When clicked:
- Scrolls to that accordion section
- Opens it
- Highlights the current stage pill
- Shows a brief "What's next:" teaser below the navigator pointing to the next stage

The navigator uses `x-data` to track `activeStage`. Clicking a stage sets `activeStage` and triggers scroll via `$refs` or `document.getElementById`.

**c) Search bar:**

Alpine.js heading filter (not full-text — per RESEARCH.md recommendation):

```html
<div x-data="guideSearch()">
  <input type="text" x-model="query" placeholder="Search topics (e.g., 'service', 'fees', 'evidence')..." 
         class="guide-search" aria-label="Search guide topics">
</div>
```

The search filters accordion sections by matching the query against each section's title and sub-heading keywords. Sections that don't match are hidden (`x-show`). When query is empty, all sections show.

Each accordion section has a `data-keywords="filing, claim, court, form 7a, fees, ..."` attribute. The Alpine search component checks if any keyword starts with the query string.

- 20-day deadline to file defence (from DEFENCE_DEADLINE_DAYS constant)

**Stage 5: Settlement Conference (Mandatory)**
- What is a settlement conference? (mandatory meeting with a judge/deputy judge to explore resolution)

- When does it happen? (court schedules it after defence filed)
- What to bring: copies of all documents, evidence list, settlement proposal
- What happens at the conference: judge reviews, identifies issues, facilitates negotiation, may make an order
- Possible outcomes: settlement, order for payment, terms of payment, referral to trial

  - "How to present": speak clearly, address the judge (not the other party), stick to facts, be brief, be honest about weaknesses
  - "Dos and Don'ts": DO bring an organized file, DO be punctual, DON'T interrupt, DON'T argue with the other party

- Present this as a recent addition (seamless integration, not flagged as disruptive change)
- What is a TMC? (Rule 16.1.01, a pre-trial conference to manage trial logistics)
- When does it happen? (after settlement conference if no resolution)

**f) Alpine.js components** (inline `<script>` at bottom):

```javascript
function guideNav() {
  return {
    activeStage: null,
    goToStage(stageId) {
      this.activeStage = stageId;
      const el = document.getElementById('stage-' + stageId);
      if (el) {
        el.scrollIntoView({ behavior: 'smooth', block: 'start' });
        // Open the accordion
        el.querySelector('[x-data]')?.__x?.$data?.open = true;
      }
    },
    get nextStage() {
      // Return the stage after activeStage
      const stages = [...document.querySelectorAll('.guide-section')];
      const idx = stages.findIndex(s => s.dataset.stage === this.activeStage);
      return idx >= 0 && idx < stages.length - 1 ? stages[idx + 1].dataset.stage : null;
    }
  };
}

function guideSearch() {
  return {
    query: '',
    matches(keywords) {
      if (!this.query.trim()) return true;
      const q = this.query.toLowerCase().trim();
      return keywords.toLowerCase().split(',').some(k => k.trim().startsWith(q));
    }
  };
}
```
