
  3. **documents/index.html** — Delete the entire `{% block head_extra %}...{% endblock %}` block (lines 5-7). This page has ONLY the HTMX tag in head_extra (no Alpine.js), so the entire block goes away. The global HTMX from base.html now covers it.
